# CyberMoe - Cybersecurity Consulting

A premium, high-performance cybersecurity consulting website built with **Svelte 5**, **GSAP**, and **Tailwind CSS**.

## Features

- **Svelte 5 Runes**: Modern reactivity and state management.
- **GSAP Animations**: Complex scroll-driven interactions, section pinning, and staggered entrances.
- **Premium Design**: Dark-mode aesthetic with magenta accents, scanline overlays, and glassmorphism.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **GitHub Actions**: Automated deployment to GitHub Pages.

## Tech Stack

- **Framework**: Svelte 5
- **Styling**: Tailwind CSS
- **Animations**: GSAP (ScrollTrigger)
- **Icons**: Lucide Svelte
- **Utilities**: clsx, tailwind-merge

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
